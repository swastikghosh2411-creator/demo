🌱 Farmer AI Assistant (AI/ML Project)

I worked on an AI-driven project called Farmer AI Assistant, aimed at supporting farmers through intelligent, data-driven insights. The system leverages machine learning and computer vision to assist in crop-related decision making. The project focuses on: Crop disease detection using image-based analysis (CNN models) Quality assessment of crops based on visual features Price prediction and trend analysis using historical market data Weather-aware recommendations to help farmers plan better The solution integrates data collection, preprocessing, and model deployment into a unified system, with the goal of making advanced AI tools accessible and practical for real-world agricultural use.
